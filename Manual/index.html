Installation

If you're reading this, you've likely already figured out how to install sm-ssc. Just in case you're reading this without having installed it yourself, however, here are the installation instructions for various operating systems.

Windows

The Windows version is available as an .exe installer and a .zip/GameData.smzip combination. The .exe installer is much easier to deal with, but in case it doesn't work out for you, we have provided a zip file for manual installation. The zip distribution can be converted into Portable mode by placing a blank Portable.ini inside of the sm-ssc directory.

Mac OS X

The Mac version is distributed as a .dmg file; Mount it, make a sm-ssc folder in Applications/, then put sm-ssc in there, and make folders as needed. That probably isn't the best way to do it, but it is A way.

Linux

Linux users are typically expected to build from source. See Compiling on the sm-ssc wiki for more information.
